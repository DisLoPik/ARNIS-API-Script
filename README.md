# ARNIS API Script
A script to automate big worlds using Arnis.
 ## What is Arnis?
 You can find out all the great things about Arnis [here](https://arnismc.com/)!
❗ ( I am not a part of Arnis )❗

## Dependencies
 1. subprocess
 2. shutil
 3. math
 4. time
 5. csv
 6. pathlib
 7. concurrent.futures
 8. threading
## How to use?
Its really simple! 😊

1. Download the "arnis_world.py"
2. Edit the paths and settings so python knows where to find your arnis exe!
	Edit lines: 33, 48, 49, 50, 57, 58. (Note you will need to create the files in lines 57 and 58)
3. Edit the rest of the settings to your liking!
4. Run the script.
5. Then you just wait. A every, very, long time. 😟
